# 🎬 Mood Movie Recommender

A React app that recommends movies based on your mood.

## 🚀 Features
- Select a mood (happy, sad, romantic, etc.)
- Fetches real movie data from TMDB API
- Displays movie posters, ratings, and descriptions

## 🛠️ Tech Used
- React
- Vite
- TMDB API

## ▶️ How to Run

```bash
npm install
npm run dev
